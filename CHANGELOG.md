# Changelog

## 1.0.1 – 2026-02-27
- Bumped Android release to `versionCode 2`, `versionName 1.0.1` for Play Store submission.
- Added in-game graphics panel (Auto/Ultra/High/Balanced/Low) with Ultra Lock default.

- Improved performance governance: manual lock option plus smarter auto quality thresholds.
- Built release artifacts:

  - AAB: `android/app/build/outputs/bundle/release/gemini-slingshot-1.0.1-release.aab`
